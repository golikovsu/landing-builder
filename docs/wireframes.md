## Design Notes

| Element           | Spec                                   |
| ----------------- | -------------------------------------- |
| Primary font      | Montserrat, 400 / 600 / 700            |
| Base text size    | 16px                                   |
| Heading sizes     | h1: 48–64px, h2: 32–40px, h3: 20–24px  |
| Border radius     | Cards: 16px, Buttons: 8px, Badges: 4px |
| Section padding   | Desktop: 96px top/bottom, Mobile: 48px |
| Max content width | 1200px centered                        |
| Card gap          | 24px                                   |
| Animation         | Fade-in on scroll, 300ms ease-out      |
